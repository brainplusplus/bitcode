# ERPNext Permission Reference

Tabel berikut mendokumentasikan 12 permission yang tersedia pada setiap **DocType** di ERPNext (Frappe Framework). Permission ini dikonfigurasi per role melalui **Role Permission Manager**.

## Permission Table

| Permission | Deskripsi | Contoh |
|------------|-----------|--------|
| **Select** | Mengizinkan user untuk mencari dan melihat nama record (di dropdown/link field), tetapi tidak dapat membuka dokumen master-nya. | User dapat memilih Customer di Sales Order, tetapi tidak dapat membuka halaman Customer master. |
| **Read** | Mengizinkan user untuk membuka dan melihat isi dokumen. | User dapat melihat Sales Invoice, tetapi tidak dapat mengubah nilai field apapun di dalamnya. |
| **Write** | Mengizinkan user untuk mengedit record yang sudah ada dan dapat diaksesnya. | User dapat mengupdate customer atau field lain di Sales Order yang sudah ada, tetapi tidak dapat membuat Sales Order baru. |
| **Create** | Mengizinkan user untuk membuat dokumen baru. | User dapat membuat Item baru, tetapi tidak dapat mengedit item yang sudah ada (memerlukan permission Write). |
| **Delete** | Mengizinkan user untuk menghapus dokumen. | User dapat menghapus dokumen berstatus Draft atau Cancelled. |
| **Print** | Mengizinkan pencetakan atau pengunduhan PDF dokumen. | Tombol Print aktif untuk user tersebut di halaman dokumen. |
| **Email** | Mengizinkan user untuk mengirim email langsung dari dokumen. | Tombol Email aktif untuk user tersebut di halaman dokumen. |
| **Report** | Mengizinkan user untuk mengakses laporan yang terkait dengan DocType tersebut. | Jika user memiliki akses ke Employee dan permission Report diaktifkan, mereka dapat melihat laporan berbasis Employee. |
| **Export** | Mengizinkan user untuk mengekspor data dari tampilan Report. | User dapat mengekspor data report ke file CSV/Excel/PDF/Doc. |
| **Import** | Mengizinkan user untuk menggunakan alat Data Import guna membuat atau memperbarui record. | User dapat mengimpor record baru atau memperbarui data yang sudah ada melalui file spreadsheet. |
| **Mask** | Mengizinkan user untuk mengaktifkan properti mask pada field tertentu dari DocType tersebut. | Jika user mengaktifkan mask pada field nomor telepon, nilainya akan ditampilkan dalam format tersembunyi (misalnya, 811XXXXXXX). |
| **Clone** | Mengizinkan user untuk melakukan clone terhadap suatu data |
